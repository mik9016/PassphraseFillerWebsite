What is Passphrase Filler?

Passphrase Filler is an extension to make your life easier by automatically filling passphrase dropdowns with specific passphrase set based on your matrix_id.


It will also automatically detect passphrase reset and replace old passphrase with new one.

User Guide

As shown on Image above we have 3 inputs to fill:


  • Environment -> not required -> just information for user on which environment we are working.
  • User -> ⚠️ required if we want to autofill feature-> check in localStorage your matrix_id value and paste it here. Should look like this -> @int-a.lacroix:matrix.messenger.bycs-int.de
  • Passphrase -> required -> 4 words divided by space. Extension is configured in the way that it looks and fills only 4 inputs.
